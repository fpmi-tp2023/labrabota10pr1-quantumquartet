
## Описание

E-commerce Wire Not App — это приложение на SwiftUI, позволяющее проходить аутентификацию и авторизацию и совершать покупки в онлайн-магазине электроники.  Проект состоит из нескольких файлов с исходным кодом. Архитектура приложения соответствует шаблону MVC. Для создания intuitive and user-friendly интерфейса было выполнено проектирование приложения с помощью онлайн-сервиса Figma. Документация проекта, включая постановку задачи и другие материалы, представлены в вики проекта. Приложение позволяет просматривать каталог товаров, добавлять товары в избранное и корзину, просматривать расположение офлайн-магазинов на карте.
